#  网页基本标签

```html
<!--meta描述搜索引擎优化的内容 -->
<meta charset="UTF-8">

<!--标题标签 h1最大-->
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>

<!--段落标签-->
<p>一段</p>

<!--换行-->
<br/>

<!--水平线标签-->
<hr/>

<!--粗体 斜体>
<strong>abc</strong>
<em>abc</em>

<!--特殊符号-->
空格:&nbsp;
大于号&gt;
小于号&lt;
版权号&copy;

```
